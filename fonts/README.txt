MAESTRO'S GAMBIT — BUNDLED DISPLAY FONTS
========================================

The game's wordmark and screen headings are set in two members of the Cinzel
family by Natanael Gama. Both are bundled here and loaded via @font-face from
css/style.css (no CDN / no network — the build stays file://-safe and offline).

  CinzelDecorative-Bold.ttf   — the ornate wordmark face (.game-title)
  Cinzel-Variable.ttf         — the refined Roman headings face (.screen-title, modal h3)
